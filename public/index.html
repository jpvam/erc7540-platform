3/5 emergency pause, 2/3 parameter changes. Signatures aggregated via ZK proof without exposing individual signer identity.
Governance Timelock
48h delay enforced on-chain for parameter changes
Fee changes, whitelist updates, and epoch modifications are timelocked. Auditors notified via Rayls Node events.
